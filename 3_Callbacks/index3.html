2.1 ASYNCHRONOUS CALLBACK

Create a function waitAndGreet that takes a name and a callback. After 2 seconds, it should call the callback with the name.


const waitAndGreet = (name, callbackHi) => {
  setTimeout(() => {
    callbackHi(name);
  }, 2000);
};

const callbackHi = (name) => console.log(`Hello ${name}!`);
waitAndGreet('Susan', callbackHi);

2.3 CALLBACKS WITH STRINGS
