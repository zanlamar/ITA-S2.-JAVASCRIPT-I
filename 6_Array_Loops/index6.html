Sprint 2.1: JavaScript

PART 6: Array Loops

👉 All the exercices results will be displayed by console.



LEVEL 1

1.1 FOR EACH

Given an array of names, use forEach to print each name to the console.


let names = ['Anna', 'Bernat', 'Clara'];
names.forEach((element) => console.log(element));

1.2 FOR-OF

Using the same array of names, print each name with a for‑of loop.


for (let element of names) {
  console.log(element);
}

1.3 FILTER

Given an array of numbers, create a new array containing only the even numbers.


let numbers = [1, 2, 3, 4, 5, 6];
let NEWnumbers = numbers.filter((element) => element % 2 === 0);
console.log(NEWnumbers);


LEVEL 2

2.1 FOR-IN

Given an object with key-value pairs, use a for-in loop to print each key and its value.


let obj = { name: 'Ona', age: 25, city: 'Barcelona' };

for (const key in obj) {
  console.log(`${key}: ${obj[key]}`);
}

2.2 FOR-OF WITH BREAK

Using an array of numbers, print the numbers until you reach 5, then stop the loop.


for (const element of numbers) {
  if (element === 5) break;
  console.log(element);
}


LEVEL 3

3.1 FOR-OF WITH INDEX

Use a for-of loop to print each element of the array and its index.


for (const [index, element] of names.entries()) {
  console.log(`${element}, ${index}`);
}
