LEVEL 1

1.1 MAP

Given an array of numbers [1, 2, 3, 4], create a new array with the square of each number.


const arrayMap = [1, 2, 3, 4];
const NEWarrayMap = arrayMap.map(element => Math.pow(element, 2));
console.log(NEWarrayMap);

1.2 FILTER

Given an array [1, 2, 3, 4], create a new array with only the even numbers.


const arrayFilter = [1, 2, 3, 4];
const NEWarrayFilter = arrayFilter.filter((element) => element % 2 === 0);
console.log(NEWarrayFilter);

1.3 FIND

Given [1, 10, 8, 11], use find to locate the first number greater than 10.


const arrayFind = [1, 10 , 8, 11];

let found = arrayFind.find((element) => element > 10);
console.log(found);

1.4 REDUCE

Given [13, 7, 8, 21], use reduce to calculate the total sum.


const arrayRed = [13, 7, 8, 21];

let resultRed = arrayRed.reduce((acc, cv) => acc + cv, 0);
console.log(resultRed);


LEVEL 2

2.1 ONE LINE FUNCTION

Given [1, 3, 7, 10, 15, 17, 11, 5, 8, 12, 9], write a one-line function that:

  • Filters numbers over and equal to 10
  • Multiplies each by 2
  • Calculates the sum of the results

const arrayOne = [1, 3, 7, 10, 15, 17, 11, 5, 8, 12, 9];

let resultOne = ((arrayOne.filter((x) => x >= 10)).map(x => x * 2)).reduce((acc, x) => acc + x);
console.log(resultOne);

3.1 EVERY / SOME

Use every and some to check if all or some values in [11, 12, 13, 14] are greater than 10.


const arrayTwo = [11, 12, 13, 14];
const over10 = (element) => element > 10;

let every = arrayTwo.every(over10);
let some = arrayTwo.some(over10);

console.log(every);
console.log(some);
