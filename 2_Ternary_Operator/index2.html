1.1 BASIC TERNARY OPERATOR

Write a function canDrive that accepts age as a parameter and uses the ternary operator to determine if the user can drive. If age is 18 or more, return 'You can drive'. Otherwise, return 'You can't drive'.


const canDrive = (age) => {
age >= 18 ? console.log(`YES, you can drive`) : console.log(`NOPE, you can't drive`);
}

canDrive(30);
canDrive(12);


1.2 COMPARISON OPERATORS

Use the ternary operator to determine which of two given numbers (num1 and num2) is greater. Return a message indicating the greater number.


const checkHigher = (num1, num2) => {
num1 > num2 ? console.log(`${num1} is higher`) : console.log(`${num2} is higher`);
}

checkHigher(92, 88);
checkHigher(33, 36);


LEVEL 2

2.1 LINKED USE OF TERNARY OPERATORS

Use linked ternary operators to determine if a number is positive, negative, or zero.


const checkPositive = (number) => {
number > 0 ? console.log(`${number} is positive`)
: number < 0 ? console.log(`${number} is negative`)
: console.log(`${number} is zero`);
};

checkPositive(92);
checkPositive(-5);
checkPositive(0);


2.2 TERNARY WITH FUNCTIONS

Create a function findTop that accepts three parameters and uses ternary logic to return the maximum value.


const findTop = (a, b, c) => {
(a > b && a > c) ? console.log(`${a} is the top number`)
: (b > a && b > c) ? console.log(`${b} is the top number`)
: console.log(`${c} is the top number`);
};

findTop(112, 91, 941);
