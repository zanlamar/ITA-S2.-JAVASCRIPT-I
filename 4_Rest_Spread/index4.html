LEVEL 1

1.1 SPREAD OPERATOR IN ARRAYS

Create two arrays, arrayA and arrayB. Use the spread operator to create a third array that contains all elements from both.


const arrayA = [4, 8, 15, 16, 23, 42];
const arrayB = [14, 28, 35, 46, 53, 62];

const arrayC = [...arrayA, ...arrayB];
console.log(arrayC);

1.2 REST OPERATOR IN FUNCTIONS

Create a function addition that uses the rest operator to accept an indefinite number of arguments and return their sum.


const addition = (...numbers) => {
  let total = 0;
  numbers.forEach(element => total += element);
  return total;
};

const numbers = [4, 8, 15, 16, 23, 42];
console.log(addition(...numbers));


LEVEL 2

2.1 COPY OBJECTS WITH SPREAD

Create an object object1. Then create a second object objecte2 as a copy of objecte1 using the spread operator. Modify one property in objecte2 and verify objecte1 is unchanged.


const object1 = {
  name: 'Limón',
  price: 1,
  km0: true
};

const object2 = {...object1};
object2.name = 'Papaya';
object2.price = 4;
object2.km0 = false;

console.log(object1);
console.log(object2);

2.2 REST IN DESTRUCTURING

Create an array with several elements. Use destructuring with the rest operator to assign the first two elements to variables, and the remaining ones to a third variable.


const arrayVarious = [1, 'a', null, 'Paco', [3, 4, 5]];
const [a, b, ...rest] = arrayVarious;

console.log(a);
console.log(b);
console.log(rest);

Create a function that accepts three arguments. Then create an array with three elements and call the function using the spread operator.


const functionSpread = (x, y, z) => x + y - z;
